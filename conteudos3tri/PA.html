Progressão Aritmética

Progressão aritmética é uma sucessão de números que seguem um padrão. É a soma de um termo inicial a1 por outro chamado razão (r).

Fórmula da progressão aritmética:

An = A1 + (n-1)*r

Fórmula da razão na progressão aritmética:

r = An - A(n-1)

Crescente - r > 0
Decrescente - r < 0
Constante - r = 0


Progressão Aritmética e Função Afim

Se formos construir um gráfico de uma progressão aritmética vamos utilizar os mesmos princípios de uma função afim, já que ela segue uma sequência em que a diferença dos termos vizinhos são constantes. Com uma diferença: NÃO PODE SER TRAÇADO UMA RETA NOS PONTOS DO GRÁFICO!


Interpolação Aritmética

Em uma sequência infinita os termos a1 e an, são chamados de extremos. Inserimos K meios aritméticos entre esses extremos. Onde n = k + 2 termos. Para encontrarmos esses termos precisamos da razão.


Soma de todos os termos

Para fazermos a soma de todos os termos de uma progressão aritmética podemos usar a seguinte fórmula:

Sn = ((A1 + An)*n ) / 2
